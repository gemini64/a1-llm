A Game is something people play (often to have fun or enjoy themselves).
There are many different types of games, including:

Sport - physical sports, often competitive
Coordination game - games that involve skill and hand-eye coordination
Board game - played on a board
Card game - played using playing cards or sometimes special cards made for the game
Video game - played on a computer, mobile device, game machine or game console. Some are online games.
Role-playing game - game where players pretend to be something or someone
