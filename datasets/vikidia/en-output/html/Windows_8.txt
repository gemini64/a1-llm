Logo of Windows 8
Windows 8 is a version of the Windows operating system developed by Microsoft. Its development began in 2009, and it was released in 2012. Windows 8 was preceded by Windows 7 and was succeeded by a minor update, Windows 8.1.
Windows 8 has a new user interface (UI) design called Metro, designed for both normal computers and touchscreen devices. It is the first Windows version to include Windows Store.
