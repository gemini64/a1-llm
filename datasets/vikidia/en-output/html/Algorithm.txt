An algorithm is a list of instructions to solve problems. It can be used in many ways.

Computers
In terms of computers, an algorithm is some code (instructions) that tells the computer how it can handle certain situations, or do certain things. This can include solving errors (problems) with programs.

Math(s)
In terms of mathematics, an algorithm is a set of instructions that a person can use to work out certain things. An algorithm could be working out the area of a triangle, height times base divided by 2, or h * b / 2.
