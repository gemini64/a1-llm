Inkscape's logo
Inkscape is a free image editing software. It can be used to edit vector image file formats, such as SVG.
Inkscape is both free in price and free in copyright. It is also open source, which means anyone can look at its source code. For these reasons, Inkscape is a popular free alternative to Adobe Illustrator.
The software is cross-platform, available for Microsoft Windows, Mac OS X, Linux, and more.
