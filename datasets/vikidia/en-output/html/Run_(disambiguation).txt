Run can have lots of different meanings, including:
Run (exercise), part of an exercise routine intended to keep a person fit and healthy.
Run (computing), a command given to a computer, and the action performed when it is executing Software.
Dye can also run, which is when it separates from fabric and mixes together, usually while doing laundry.
A run can also happen to a bank, which is when a lot of people try to withdraw their money from banks belonging to the same company, at the same time.
