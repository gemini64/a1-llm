La Microsoft Corporation è un'azienda statunitense operante nel settore dell'informatica. È famosa per aver ideato il sistema operativo Microsoft Windows, il sistema operativo MS-DOS, le console di gioco Xbox e la suite per ufficio Microsoft Office.

Storia
L'azienda viene fondata nel 1975 da Bil Gates e Paul Allen.

Prodotti

Microsoft Windows
Xbox
Microsoft Office
OneDrive
Outlook.com
MS-DOS
Visual Studio



Critiche
La Microsoft è stata a lungo criticata per i suoi comportamenti definiti "anti-concorrenziali" cioè scorretti verso le aziende che vendono un prodotto simile per vendere di più (*) .
