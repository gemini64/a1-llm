L'HTML è un linguaggio markup e la sua sigla significa HyperText Markup Language. Questo linguaggio viene utilizzato per definire la struttura e gli elementi di una pagina web. Usato insieme a CSS e a JavaScript può creare pagine con effetti grafici e dinamiche. Attualmente è giunto alla sua versione 5. È stato pubblicato inizialmente nel giugno del 1993 (*) .
La sua ultima versione può sostituire Adobe Flash. Anche Adobe stessa, società che sviluppa Flash, ha consigliato di utilizzare l'HTML. (*)

Esempi


<html>
<head>
<title> Il mio sito web </title>
<meta charset="UTF-8">
</head>
<body>
<h1> Benvenuto nel mio sito web </h1>
</body>
</html>
Questo esempio di codice farà apparire nel corpo della pagina la scritta Benvenuti nel mio sito web con il font dell'h1 (elemento HTML), modificabile poi con il CSS. Nel titolo della scheda, in alto, apparirà invece ciò che è scritto all'interno dell'elemento title, ovvero Il mio sito web.


HTML in applicazioni desktop
L'HTML è nato per essere usato nel web ma attraverso Electron può essere utilizzato anche nella creazione di applicazioni desktop o software.
