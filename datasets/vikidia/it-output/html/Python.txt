Il logo di Python.
Python è un linguaggio di programmazione ad alto livello, cioè vicino al parlato delle persone più che a quello del computer.
Un programma in Python  può girare su qualsiasi computer con installato l'interprete per Python.

Esempi


print("Ciao")

Scriverà a monitor "Ciao"
Questo linguaggio è stato creato nel 1991 da Guido van Rossum, un informatico olandese, e il nome è ispirato ai Monty Python, un gruppo comico britannico.
Il suo sviluppo è gestito dalla Python Software Foundation, e la versione attuale di Python è la 3.8.0b1 (4 giugno 2019). Python è un software libero cioè si può scaricarlo, modificarlo e distribuirlo gratuitamente.
